# Gabi_JogoDaVelha
Desenvolvi esse Jogo da Velha para treinar meus conhecimentos sobre React

Este Projeto foi desenvolvido por: Gabriela Paulini Deutner
Agradecimentos: Eu me inspirei em tutoriais do Professor Marco Bruno da Collabcode - GitHub: @marcobrunodev

Instalar pacote npm: npm install (diretório "jogo-da-velha" onde está o package.json - executar cd .\jogo-da-velha\)
Iniciar a aplicação: npm start
Finalizar a aplicação: ctrl + c
